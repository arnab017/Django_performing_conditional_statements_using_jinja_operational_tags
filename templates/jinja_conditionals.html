Jinja Operational Tags - Conditional statements

if condition using jinja operational tags

{% if a > 18 %}

{{a}}(a) is greater than 18

{% endif %}

if-else condition using jinja operational tags

{% if a > b %}

{{a}}(a) is greater than {{b}}(b)

{% else %}

{{b}}(b) is greater than {{a}}(a)

{% endif %}
